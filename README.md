# Physical AI & Humanoid Robotics Book

This is a comprehensive course for senior undergraduate AI students covering the fundamentals of physical AI and humanoid robotics.

## Setup

1. Make sure all dependencies are installed:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

To build the website for production:

```bash
npm run build
```

## Deployment

To deploy to GitHub Pages:

```bash
npm run deploy
```

## Course Structure

The course is organized into the following modules:

1. Introduction and Learning Outcomes
2. The Robotic Nervous System (ROS 2)
3. Digital Twin (Gazebo & Unity)
4. AI-Robot Brain (NVIDIA Isaac)
5. Vision-Language-Action (VLA)
6. Capstone Project: Autonomous Humanoid

Each module includes concepts, architecture diagrams, hands-on examples, and mini exercises.
